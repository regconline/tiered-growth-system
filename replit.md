# REGC Digital — Astro Site

Healthcare digital marketing website for South African medical practices.

## Stack

- **Astro 4** — static site generator
- **React 18** — used only for interactive islands (Navbar, Contact form, WhatsApp widget, BackToTop)
- **Tailwind CSS** + `@tailwindcss/typography` + `tailwindcss-animate`
- **MDX / Content Collections** for the blog
- **Sitemap** auto-generated by `@astrojs/sitemap` → `/sitemap-index.xml`

## SEO

- `BaseLayout.astro` emits per-page `<title>`, `<meta description>`, canonical, `hreflang en-ZA + x-default`, full Open Graph + Twitter cards (with `og:image:alt/width/height` and `og:locale`), `theme-color`, and a robots directive (noindex toggle).
- Sitewide JSON-LD is injected on every page: `Organization` (with `contactPoint`, `address`, `sameAs`) and `WebSite`. Pages opt in to additional schemas via the `jsonLd` prop.
- Per-route schemas:
  - Home: `ProfessionalService` + `FAQPage`
  - `/services/`: `BreadcrumbList` + `ItemList`
  - `/services/{slug}`: `Service` (with `AggregateOffer` priced in ZAR) + `BreadcrumbList`
  - `/domains/`: `BreadcrumbList` + `ItemList`
  - `/domains/{slug}`: `ProfessionalService` + `BreadcrumbList` + `FAQPage` (when FAQs exist)
  - `/locations/`: `BreadcrumbList` + `ItemList`
  - `/healthcare-marketing-{loc}`: `LocalBusiness` (priceRange `$$`) + `BreadcrumbList`
  - `/blog/`: `BreadcrumbList` + `Blog`
  - `/blog/{slug}`: `BlogPosting` (with `inLanguage`, `articleSection`, `keywords`, `image`) + `BreadcrumbList`
  - `/about/`, `/contact/`: `BreadcrumbList` + `AboutPage`/`ContactPage`
- Helper: `src/lib/seo.ts` (`breadcrumbList`, `itemList`).
- `public/robots.txt` allows all by default, blocks `/admin/`, `/private/`, `/thank-you/`, throttles AhrefsBot/SemrushBot, and blocks GPTBot/CCBot. Points to the auto-generated `/sitemap-index.xml`.
- Site constants (`SITE` in `src/data/site.ts`) hold canonical URL, contact details, default OG image, locale, region, theme color, and `sameAs[]` (currently empty — populate as social profiles go live).
- Default OG image is the logo (`/assets/regc-logo.png`); replace with a 1200×630 social card at `public/og-default.jpg` when ready and update `SITE.defaultOg`.

## URLs Preserved

`/`, `/about/`, `/services/`, `/services/{slug}/`, `/domains/`, `/domains/{slug}/`, `/contact/` — plus new `/blog/` and `/blog/{slug}/`.

## Workflow

`Start application` runs `npm run dev` which executes `astro dev --host 0.0.0.0 --port 5000` (webview).

## Build / Deploy

- Build: `npm run build` → outputs static files to `./dist`
- Deployment: configured for `static` target serving `./dist` (see `.replit`)

## Conventions

- All design tokens live in `src/styles/global.css` as HSL CSS vars. Tailwind config consumes them via `hsl(var(--foo))`.
- React islands use `client:load` for interactive-on-mount, `client:idle` for non-critical widgets.
- Blog posts: front-matter must satisfy the Zod schema in `src/content/config.ts`.
- Adding a new blog post = just drop a `.md` or `.mdx` file in `src/content/blog/`.
