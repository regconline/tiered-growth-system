## ISSUE #1 — CRITICAL: Fix 307 Temporary Redirects → Change to 301 Permanent Redirects

### Problem
The crawl identified **187 URLs** with `307 Temporary Redirect` responses. This includes:
- All **canonical link** declarations pointing from `www.` to non-`www.` versions that are returning 307
- All **sitemap.xml entries** pointing to non-`www.` URLs that return 307
- The root HTTP redirect `http://regcdigital.co.za/ → https://regcdigital.co.za/` returning 308 but the chain continues with a 307

307 redirects tell search engines the redirect is **temporary**, meaning Google will not pass PageRank through them, will not consolidate canonical signals, and may continue indexing both versions. This is the **highest priority SEO issue on the site.**

### Root Cause
The site has two canonical domains in conflict:
- Pages load on `www.regcdigital.co.za` (200 OK) ✅
- Canonical tags point to `regcdigital.co.za` (non-www) which then 307-redirects to `www`
- Sitemap lists `regcdigital.co.za` (non-www) URLs

### Fix Required

**Step 1: Standardise on `www.regcdigital.co.za` as the canonical domain.**

In `next.config.js` (or `next.config.mjs`), ensure the following redirect is in place:

```js
async redirects() {
  return [
    {
      source: '/:path*',
      has: [{ type: 'host', value: 'regcdigital.co.za' }],
      destination: 'https://www.regcdigital.co.za/:path*',
      permanent: true, // This generates a 301, not 307
    },
  ];
},
```

**Step 2: Fix all canonical tags.**

Every page currently has a canonical pointing to `https://regcdigital.co.za/[slug]`. Change all canonicals to point to `https://www.regcdigital.co.za/[slug]`.

In your shared metadata layout or `generateMetadata()` function, ensure:

```js
alternates: {
  canonical: `https://www.regcdigital.co.za/${slug}`,
},
```

Replace any hardcoded `regcdigital.co.za` (without `www`) in canonical generation with `www.regcdigital.co.za`.

**Step 3: Fix sitemap.xml**

Update `sitemap.ts` (or `sitemap.xml`) to use `www.regcdigital.co.za` for all URLs:

```ts
// In app/sitemap.ts
export default function sitemap(): MetadataRoute.Sitemap {
  const baseUrl = 'https://www.regcdigital.co.za'; // NOT regcdigital.co.za
  // ...
}
```

**Step 4: Check hosting/server redirects**

If using Vercel, check `vercel.json` for any redirect rules. If using Nginx or another server, ensure the non-www → www redirect uses `301` not `307`. The `307` is often caused by a misconfigured server redirect or a missing `permanent: true` flag.

### How to Implement Keywords in Next.js

#### 1. Update `generateMetadata()` on each location page

```tsx
// Example: app/healthcare-marketing/[location]/page.tsx
export async function generateMetadata({ params }): Promise<Metadata> {
  const { location } = params;
  const locationData = getLocationData(location); // your data source

  return {
    title: `Healthcare Marketing ${locationData.city} | REGC Digital`,
    description: `Expert healthcare digital marketing in ${locationData.city}, ${locationData.province}. We help doctors, dentists, physios & allied health professionals attract more patients. HPCSA-aligned strategies.`,
    keywords: locationData.keywords.join(', '), // add from keyword sets above
    alternates: {
      canonical: `https://www.regcdigital.co.za/healthcare-marketing-${location}/`,
    },
    openGraph: {
      title: `Healthcare Marketing ${locationData.city} | REGC Digital`,
      description: `Grow your practice in ${locationData.province} with tailored digital marketing.`,
    },
  };
}
```

#### 2. Add JSON-LD LocalBusiness Schema to Location Pages

Add this schema to every location page to boost local SEO signals:

```tsx
const jsonLd = {
  '@context': 'https://schema.org',
  '@type': 'LocalBusiness',
  name: 'REGC Digital',
  url: 'https://www.regcdigital.co.za',
  telephone: '+27640826855',
  areaServed: [
    {
      '@type': 'AdministrativeArea',
      name: locationData.province, // e.g. "KwaZulu-Natal"
    },
    {
      '@type': 'City',
      name: locationData.city, // e.g. "Durban"
    },
  ],
  description: `Healthcare digital marketing agency serving ${locationData.city} and the ${locationData.province} province.`,
  serviceType: [
    'Healthcare Marketing',
    'Medical SEO',
    'Healthcare Website Design',
    'Patient Acquisition',
    'Google Ads for Doctors',
    'Healthcare Content Marketing',
  ],
};

// In your component JSX:
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
/>
```
